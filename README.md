# portal-backport
MC 1.21+传送门特性移植（事实上还有TNT相关修复）  
**目前移植的功能：**  
- 允许下落的方块穿过地狱门
- 允许点燃的tnt实体穿过地狱门
- 穿过地狱门的tnt不会炸毁地狱门
- 穿过地狱门的tnt保留激活它的实体信息
- 穿过地狱门的tnt保留传送标志到Teleport字段（重载带有穿过地狱门的TNT的区块，导致TNT可以炸毁地狱门问题修复）
- 使得tnt保留点燃它的实体的uuid到Owner字段（因为底层机制原因，此修复为不完全修复，不保证TNT重新从存档加载后能正常找到实体）

**明确不进行移植的功能：**  
- 骑乘实体传送（有大量破坏性更改和注入）
- 非玩家实体进入没有对向维度可传送的地狱门时，也能像玩家一样创建对向维度的新地狱门（这个特性会导致许多带传送门的机器出问题，不予移植）
